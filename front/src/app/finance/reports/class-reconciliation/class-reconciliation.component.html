info What does class reconciliation do?

  • Runs full finance reconciliation for every student enrolled in the selected class and term.
  • Corrects overpayments, verifies balances, applies credit, and reallocates receipts to invoices.
  • The report shows success/failure per student and can be downloaded as JSON.
